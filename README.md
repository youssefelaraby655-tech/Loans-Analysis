 📝 Project Overview
An interactive banking dashboard analyzing loan portfolios, credit risk indicators, and customer repayment behavior to evaluate operational efficiency and portfolio health using Power BI, SQL, and financial modeling.

🎯 Business Objectives
* 📉 **Credit Risk Assessment:** Identify key indicators of loan defaults and segment borrowers by risk profiles.
* 💰 **Portfolio Performance:** Track total disbursed loans, interest revenue, and Non-Performing Loans (NPL) ratios.
* ⚡ **Operational Efficiency:** Analyze banking behavior and transaction amounts across different distribution channels.

🛠️ Tech Stack & Tools
* 🧼 **Power Query / SQL (T-SQL):** Data extraction, heavy data cleaning, and relational schema preparation.
* 🏗️ **Power BI:** Relational data modeling (Star Schema) to optimize query performance.
* 🧮 **DAX / Advanced Excel:** Advanced financial measures, risk percentages, and time-intelligence KPIs.
* 📊 **Power BI Desktop:** Dynamic, executive-level interactive dashboards.

 📊 Key Insights & Features
* 💳 **Loan Portfolio Summary:** High-level tracking of total active loans, total funded amounts, and received payments.
* ⚠️ **Risk & Default Matrix:** Analysis of bad loans vs. good loans, default rates by loan grade, and employment length.
* 🗺️ **Demographic & Purpose Analysis:** Visualizing loan distribution by purpose (e.g., debt consolidation, home, small business) and geographic regions.
* 📅 **Repayment Trends:** Monthly monitoring of payment streams and historical credit behavior patterns.

📂 Project Files
* 📊 **Loans.pbix:** The main interactive Power BI dashboard containing the banking analytical models and visual reports.
